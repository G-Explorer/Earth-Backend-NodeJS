## Introduction

Earth-Backend(version3-inside) is a photography website built by NodeJS and MySQL

## Main Requirements

* node: `^8.9.0`
* mysql: `^5.7.0`

## Quick start

1. ENV: development (feature and dev branch)
```shell
npm run dev
```

3. ENV: sandbox (release branch)
```shell
npm run sandbox
```

4. ENV: production (master branch)
```shell
npm run prod
```
